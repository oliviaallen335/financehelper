Return on an Investment
Periodic return
- Return over a specific period
- Can be averaged to understand over time
Holding period return
- Return over an entire range of periods
- Simple percentage change from start to end of investment
Geometric average return
- Return implied by compounding over an entire range of periods
- Found using TVM equations

How to Think About Risk?
Risk can be defined as a measure of the uncertainty in a set of potential outcomes for an event i.e., how those outcomes differ from what is expected.
- “Risky” implies not only the potential for loss but also for uncertain gain (risk does not take sides)
- Risk is an ex-ante measure
- Measure the level of risk through expected value and standard deviation.
